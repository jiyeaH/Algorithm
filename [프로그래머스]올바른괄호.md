- 난이도 : Level 2

## 2. 문제 해결

- 코드1 (스택이용)
```python
def solution(s):
    answer = True
    temp = []
    s = list(s)
    
   
    for i in s:
        #temp에 '('를 push함
        if i =='(':
            temp.append(i)
        
        #temp에서 '('와 짝이 있는 ')'가 있다면 pop
        else: 
            if len(temp) != 0:
                temp.pop()
            else:
                answer = False
                break  
   
   if len(temp)!= 0:
        answer = False
    
    return answer
```
