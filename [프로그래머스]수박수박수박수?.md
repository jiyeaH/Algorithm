# 프로그래머스  

## 1. 문제 정보

- 출처 : 수박수박수박수?

- 난이도 : Level 1

## 2. 문제 해결

- 코드1 
```python
def solution(n):
    answer = ''
    for i in range(1, n+1):
        if i%2 == 0:        
            answer = answer + "박"
        else:               
            answer = answer + "수"
    return answer
```
