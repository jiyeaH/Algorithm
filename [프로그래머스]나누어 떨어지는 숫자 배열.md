## 1. 문제 정보

- 출처 : 나누어 떨어지는 숫자 배열

- 난이도 : Level 1

## 2. 문제 해결

- 코드1 
```python
def solution(arr, divisor):
    answer = []
    
    for i in range(len(arr)):
        if arr[i] % divisor == 0:
            answer.append(arr[i])
    answer.sort()
    if not answer:
        answer.append(-1)
        
    return answer
```
