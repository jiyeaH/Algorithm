## 2. 문제 해결

- 코드1    
맨처음 binary search 문제인줄 알고 계속 고민하다가 쉽게 생각해보기로 했더니 그냥 풀렸다,,,   
```python
def solution(d, budget):
    answer = 0
    d.sort()
    cnt = 0
    for i in d:
        if budget < i:
            break
        budget -= i
        cnt +=1
    answer = cnt
    return answer
```
