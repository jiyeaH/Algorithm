## 1. 문제 정보

- 출처 : 자릿수 더하기
- 난이도 : Level 1

## 2. 문제 해결

- 코드1
```python

def solution(n):
    answer = 0
    
    #n을 문자열로 변환하여 num에 저장
    num = str(n)
    
    #문자열에 저장된 숫자들을 int형으로 다시 변환하여 answer에 더함
    for i in range(len(num)):
        answer = int(num[i]) + answer

    return answer
```
