## 2. 문제 해결

- 코드1 
```python
def solution(arr):
    
    answer = []
    temp = -1
    
    #temp와 arr의 원소가 다르면 그 원소를 temp에 저장
    #answer에 temp를 추가해준다.
    for i in arr:
        if i != temp:
            temp = i
            answer.append(temp)
    return answer
```
