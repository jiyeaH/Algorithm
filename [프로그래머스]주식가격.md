## 1. 문제 정보

- 출처 : 주식 가격

- 난이도 : Level 2

## 2. 문제 해결

- 코드1 (주식이 내려가는 시점last를 구해 첫 시점 i를 빼준다)
```python
def solution(prices):
    answer = []
    for i in range(0,len(prices)):
        last = len(prices) - 1  
        for j in range(i + 1, len(prices)):
            if prices[i] > prices[j]:
                last = j
                break
        answer.append(last - i)
    return answer
```
